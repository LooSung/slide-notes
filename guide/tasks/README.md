# 작업 카드

긴 발표자료를 만들 때 작업을 잘게 나누는 카드입니다.
카드는 언제든 수정 가능하며 프로젝트 `guide/tasks/` 에 둡니다.

## 사용법

1. 발표자료 생성: `npm run create -- my_presentation`
2. 해당 작업 카드를 `decks/my_presentation/docs/tasks/` 에 복사하거나 링크
3. Cursor / Claude / Codex에 카드 전체 붙여넣기
4. 카드의 완료 조건을 채운 뒤 다음 작업으로 이동
