# Task 카드

하네스 Phase마다 **한 세션에 하나**만 수행한다.  
카드는 언제든 수정 가능 — 레포 `docs/tasks/` 에 둔다.

## 사용법

1. 덱 생성: `npm run create -- my_deck`
2. 해당 Task 카드를 `decks/my_deck/docs/tasks/` 에 복사하거나 링크
3. Cursor / Claude / Codex에 카드 전체 붙여넣기
4. **Done when** 충족 후 다음 Task
