# slide-notes — Claude Code

발표자료: **`decks/<이름>/docs/`** 에 원고 · **`slides/`** 에 슬라이드 화면.

## 읽을 것

- [guide/START_HERE.md](./guide/START_HERE.md) — 따라하기
- [guide/HARNESS.md](./guide/HARNESS.md) — 작업을 나눌 때
- [guide/SLIDE_HTML_RULES.md](./guide/SLIDE_HTML_RULES.md) — 슬라이드 화면을 만들 때

`guide/` = 도구 설명. `decks/*/docs/` = 작업 원고.
슬라이드 화면은 `slide-map.md`의 현재 행과 해당 장별 원고만 읽어 만든다. 긴 발표자료에서 원고와 예시를 한 번에 읽지 않는다.

## 완료 시 안내

```bash
npm run open -- <발표자료_이름>
```
