# 유저 생성 기능


## 개요

- **기능**: 새로운 유저 계정 생성

- **엔드포인트**: `POST /api/internal/accounts/user`


## 핵심 기능 (Requirements)

1. 이메일 중복 체크 <!-- 가장 중요 -->

2. 비밀번호 검증 (최소 8자 이상)

3. 비밀번호 해싱 (`bcrypt` 사용)

4. 유저 생성 (UUID 발급)

5. MongoDB 저장


## 플로우 (Flow)

`요청` → `검증` → `이메일 중복 체크` → `비밀번호 해싱` → `저장` → `응답`


## 아키텍처

- **Service**: `UserCreationService`

- **Repository**: `UserRepository` (Interface)

- **Infrastructure**: `UserRepositoryImpl` (MongoDB)
