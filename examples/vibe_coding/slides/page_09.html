User 유저 생성 기능을 만들어줘.
필드는 이메일, 비밀번호, 이름이 필요해.
비밀번호는 bcrypt로 해싱해서 저장해줘.

AI Assistant 네, 요청하신 내용대로 유저 생성 서비스 로직을 제안합니다.

async def create_user(email, password, name):
# 1. 비밀번호 해싱
hashed_pw = bcrypt.hashpw(
password.encode(),
bcrypt.gensalt()
)
# 2. 유저 객체 생성 및 저장
user = User(
email=email,
password=hashed_pw,
name=name
)
return await repo.save(user)
